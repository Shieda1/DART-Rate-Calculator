What is the DART rate calculator?
The DART rate calculator is a tool that calculates the rate of injuries that lead to time off for recovery in a company.
How to calculate DART rate?
To calculate DART rate you should use the following formula:
DART rate = (total number of DARTs * 200000) / total hours worked
